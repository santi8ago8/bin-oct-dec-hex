bin-oct-dec-hex, not tested
===========================

bin-oct-dec-hex conversor in JavaScript

use:  
Convert decimal to bin,oct,hex
------------------------------
```
toN(329,2) //to bin
toN(329,8) //to oct
toN(329,16) //to hex
```
Convert bin,oct,ext to decimal
------------------------------
```
toDecimal(329,2) //to dec
toDecimal(329,8) //to dec
toDecimal(329,16) //to dec
```
